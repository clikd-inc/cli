# Quick Start

Get your first changelog generated in under 5 minutes.

## Step 1: Install CLIKD

<Tabs items={['macOS', 'Windows', 'Linux']}>
<Tab value="macOS">
```bash
# Using Homebrew
brew tap clikd-inc/tap
brew install clikd
```
</Tab>

<Tab value="Windows">
```bash
# Using Scoop
scoop bucket add clikd https://github.com/clikd-inc/scoop-bucket
scoop install clikd
```
</Tab>

<Tab value="Linux">
```bash
# Download and install
curl -L https://github.com/clikd-inc/cli/releases/latest/download/clikd_linux_amd64.zip -o clikd.zip
unzip clikd.zip && chmod +x clikd && sudo mv clikd /usr/local/bin/
```
</Tab>
</Tabs>

**Verify installation:**
```bash
clikd version
```

## Step 2: Quick Setup

Navigate to your Git repository and run:

## Troubleshooting

**Command not found?**
```bash
# Check if CLIKD is in PATH
which clikd
```

**API key issues?**
```bash
# Test your setup
clikd ai-test "Hello"
```

**No commits found?**
```bash
# Make sure you're in a Git repository
git status
```
